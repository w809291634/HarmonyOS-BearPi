* 说明：人体感应协议
* 协议说明：
*    属性         参数          权限    说明
*    人体红外     A0            R       布尔型，0~1，1：监测到有人，0：未监测到有人

*    上报使能     D0            R       D0的bit位控制上报状态，bit0~bit3分别对应A0~A1：0表示不允许上报，1表示允许上报
*                OD0/CD0       RW      对D0的位进行操作，CD0表示位清零操作，OD0表示位置一操作
*
*    控制编码     D1            R       D1的bit位控制设备：
                                          Bit0表示蜂鸣器开关状态，0-关闭，1-打开;
*                OD1/CD1       W       对D1的位进行操作，CD1表示位清零操作，OD1表示位置一操作
*
*    上报间隔     V0            RW      设置为传感器上报时间间隔，单位秒
